gmeteor
============
helper [meteor](https://www.meteor.com/), [mup](https://github.com/arunoda/meteor-up) and [gulp](http://gulpjs.com/)

# install
```
npm install -g gmeteor
```

# command
## run
```
gmeteor run
gmeteor run android
gmeteor run ios
```

## build
```
gmeteor build android
gmeteor build ios
```

## deploy
```
gmeteor deploy
```

## gulp wrapper
```
gmeteor gulp [tasks...]
```

## meteor wrapper
```
gmeteor meteor [command]
```

## mup wrapper
```
gmeteor mup [command]
```
